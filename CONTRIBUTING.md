# Rules for adding applications

## Validate rules

1. Application must be multilanguage or with English primary language.
2. Applications that configure specific hardware cannot be added (Miro, Dygma Keyboard Configurator).
3. All added applications must be in active development status (With code changes over the last year)
4. Applications from closed distribution sources are not allowed (applications received in subscription for ko-fi or applications from itch.io, if they are paid)

## How to choose correct category

1. If you are adding an app that helps improve the productivity of the team or your own (To-do lists, Agenda, Kanban boards) then you need to add a **"Productivity"** category.
2. If you add a messenger or a VPN, Proxy client, you must designate the category **"Network"**
3. If you add a terminal or file manager, you must specify the **"System"** category, because no OS can live without them
